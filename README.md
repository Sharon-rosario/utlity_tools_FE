# $1 Utility Tools Platform Frontend

This is a proof-of-concept frontend for the **$1 Utility Tools Platform** designed to showcase the vision to investors and team members. 

## The Concept
A platform offering single-use, high-value utility tools designed to save users hours or days of work for just $1. 
No subscriptions, no ongoing commitments—just pay for what you use.

## Built With
- React
- Vite
- TypeScript
- Vanilla CSS (Dark/Glow Theme)

## Getting Started
To run this project locally:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server

## Strategy Overview
The platform encompasses:
- A clear **$1 mental model** for low barrier to entry
- **SEO-led growth** driven by single-purpose utility pages
- **Short-form video distribution** (Reels, TikTok, Shorts)
- A **Referral loop** to drive organic sharing
